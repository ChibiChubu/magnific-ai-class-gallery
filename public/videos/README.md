# Video Folder

Letak fail video galeri dalam folder ini.

Contoh:

- `prompt-image-cinematic.mp4`
- `storyboard-moodboard.mp4`
- `spaces-magnific-workflow.mp4`
- `upscale-enhance-final-output.mp4`

Dalam React, panggil video sebagai:

```jsx
src: '/videos/prompt-image-cinematic.mp4'
```

Format paling senang untuk browser ialah `.mp4` dengan codec H.264.
